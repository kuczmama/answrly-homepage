What does it do?

It shows a countdown timer on your cart page. This can you increase conversions.

Manual Installation Instructions

If you have installed the app, and it works, you are done!

But it is possible you may need to manully install the theme.

  1. Go to the theme.liquid file.
  2. Paste the below code directly above the </head> tag
    3. <script src="https://cdn.answrly.com/hurry-up-countdown-timer/hurry-up-countdown-timer.js?shop={{your shop}}.myshopify.com"></script>

    Replace {{your shop}} with your shop URL

  3. Done!

If you have any issues getting the app working, please email support@answrly.com.

What is the coupon code for?

Occasionaly, we want to be able to track where installs have come from, we use different coupon codes to do so. If you install the app with a coupon code, you may get discounts or other deals.

The timer isn't showing up

Make sure that your cart selector is showing up properly by default it is `form.cart`, but you can change it to wherever you want the timer to show up above. This is the css selector for your cart. If you need help doing this feel free to reach out to us, and we will help you get it set up.

How do I uninstall?

Click on the "Apps" page in the sidebar in shopify, and click on the uninstall button. Everything will uninstall automatically, there is nothing else you need to do.
